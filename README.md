**How to Play**



The project is playable by downloading the .py file and pressing the play icon. You will be prompted to enter a code by the host of the game -- the 'executioner'



You have 6 chances ('lives') to guess the word.



You are prompted by the executioner to enter a word or letter. Please type either 'word' or 'letter' in full, and then proceed to make your guess. 



If you guess incorrectly, a life will be docked. If you use all 6 lives, then you lose. If you guess the word before you use all 6 lives, then you win!



**Additional comments**



For the submitted version of the game, there is only one word that you can guess from. However future updates to the game will include an expanded word library, and will make use of the 'random' function.
